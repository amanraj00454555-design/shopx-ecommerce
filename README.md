# ShopX — E-Commerce Web App

A full-stack e-commerce application built with Node.js, Express, MySQL and vanilla JavaScript.

## Features
- User & Admin role system
- Product management (add, edit, delete)
- Shopping cart & order placement
- JWT authentication
- Responsive dark UI

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MySQL
- **Auth:** JWT + bcrypt

## Setup
1. Clone the repo
2. Run `cd backend && npm install`
3. Create `.env` file with `PORT=5000` and `JWT_SECRET=yoursecret`
4. Start MySQL and create database `ecommerce_db`
5. Run `node index.js`
